features

Everything between commit and production.

[ Zero config ]

Detects your stack and writes the pipeline for you. Override only what you need.

[ Parallel builds ]

Content-addressed cache and a remote build farm. Cold builds are the exception.

[ Instant rollback ]

Every deploy is immutable and addressable. Roll back a bad ship in one command.

[ Edge deploys ]

Push artifacts to dozens of regions with canary and blue-green built in.

[ Secrets vault ]

Encrypted at rest, injected at runtime, never printed in a log line.

[ Audit logs ]

Every build, deploy, and rollback is signed and queryable. Own the trail.

how it works

Three commands. No control plane to babysit.

01

Install

One script, no daemon. Works on your laptop and in CI unchanged.

02

Init

Point ANVIL at a repo. It infers the build, test, and deploy graph.

03

Deploy

Ship to the edge with canaries. Watch it, and roll back if it blinks.
